# Контент-мастер PRO — Quick Start
**3 промпта для первого запуска. Запускайте прямо сейчас.**

---

## Прежде всего: заполните 5 полей в config.yaml

```yaml
business:
 name: "Ваша компания / личный бренд"
 niche: "онлайн-образование"    # ← ваша ниша
 product: "курс по [теме]"     # ← ваш продукт
channels:
 telegram:
  channel_link: "@ваш_канал"    # ← ваш TG-канал
tone_of_voice:
 style: "деловая прямая"      # ← стиль голоса бренда
```

Это займёт 3-5 минут и сделает все 10 режимов персонализированными.

---

## Промпт 1: Контент-стратегия (первый запуск)

**Скопируйте и вставьте целиком:**

```
стратегия контента для [ваша ниша]: 
[2-3 предложения о вашем бизнесе, продукте и аудитории]

Площадки: Telegram + Instagram
Цель на ближайшие 3 месяца: [рост подписчиков / продажи / узнаваемость]
```

**Пример заполненного:**
```
стратегия контента для онлайн-школы кулинарии:
Обучаю домашних кулинаров готовить блюда ресторанного уровня.
Аудитория — женщины 28-45 лет, хотят готовить красиво и вкусно.
Продаём курсы от 3 900 до 18 000 руб.

Площадки: Telegram + Instagram
Цель на ближайшие 3 месяца: рост подписчиков Telegram с 800 до 2000
```

**Что получите:**
- Анализ ниши и аудитории
- Позиционирование в контенте
- Полный Tone of Voice с примерами правильного/неправильного
- Контент-микс (пропорции обучающего/продающего/вовлекающего/личного)
- KPI стратегии

## Промпт 2: Контент-план на месяц

**Запустите сразу после стратегии:**

```
контент-план на [месяц, год] для [ниша], площадки: Telegram + Instagram + email
Контент-цель: [цель]
```

**Пример:**
```
контент-план на май 2026 для онлайн-школы кулинарии,
площадки: Telegram + Instagram
Контент-цель: прогрев аудитории к запуску курса 20 мая
```

**Что получите:**
- Таблица 30 публикаций: дата / площадка / формат / тема / этап воронки / цель / метрика
- Привязка TOFU/MOFU/BOFU к неделям
- Примечания по инфоповодам и вечнозелёному контенту

## Промпт 3: Первый пост — сторителлинг

**Попробуйте самый конвертящий формат:**

```
напиши сторителлинг про [конкретный момент из вашего опыта]:
[1-2 предложения о ситуации/истории]
Площадка: Telegram
TOV: [ваш стиль — прямой/живой/экспертный]
```

**Пример:**
```
напиши сторителлинг про момент когда мой первый онлайн-курс купили 47 человек за 3 дня:
Я выложила анонс без прогрева, просто в лоб — "есть курс", и ждала. Думала провал.
Площадка: Telegram
TOV: живой, личный, без пафоса
```

**Что получите:**
- Готовый пост 500-900 символов для TG
- Структура: крючок → контекст → конфликт → путь → развязка → мораль → CTA

## Следующие шаги после первых трёх промптов

1. `анализ конкурентов в [ниша]` — найдите незанятые темы за 5 минут
2. `SEO-ядро для [ниша]` — 200+ ключей для органического трафика
3. `рерайт под форматы: [вставьте лучший пост]` — из одного текста 5 форматов
4. `воронка контента для [продукт]` — TOFU/MOFU/BOFU с таблицей тем
5. `email-цепочка: welcome для [ниша]` — 5 писем для новых подписчиков
