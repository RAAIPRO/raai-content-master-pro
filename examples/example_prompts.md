## Быстрый запуск по режимам

### 1. Контент-стратегия
```
стратегия контента для [ниша]: [описание бизнеса 2-3 предложения]
```

### 2. Контент-план на месяц
```
контент-план на [месяц] для [ниша], площадки: TG + Instagram + email
```

### 3. SEO-ядро
```
SEO-ядро для [ниша], регион: [город/страна]
```

### 4. Пост по формату
```
напиши [кейс/экспертный/продающий/сторителлинг/обучающий/провокация/личный] про [тема]
```

### 5. Рерайт → 5 форматов
```
рерайт под форматы: [вставить текст]
```

### 6. Анализ конкурентов
```
анализ конкурентов: [ниша или @ник, @ник, @ник]
```

### 7. A/B заголовки
```
5 заголовков для: [тема], аудитория: [описание]
```

### 8. Контент-воронка
```
воронка контента для [продукт], цена [N] руб, аудитория: [описание]
```

### 9. Отчёт по контенту
```
отчёт контент за [месяц]: TG [подписчиков] [постов] ER [%], IG [подписчиков] [постов] ER [%], email [база] open rate [%] CTR [%]
```

### 10. Email-цепочка
```
email-цепочка: [welcome/nurture/продажа/реактивация] для [ниша], лид-магнит: [название]
```
